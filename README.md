# Chip-8 emulator
A Chip-8 emulator written to learn about instruction sets and computer architecture.

## Timing
- display: ~60hz
- cpu execution: ~500hz

## Keyboard Layout
1 2 3 4  <br>
Q W E R  <br>
A S D F  <br>
Z X C V
